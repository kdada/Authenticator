# 2FA 验证器

双因子认证（2FA）密码生成工具，支持 GitHub，Microsoft，Google 等站点的标准令牌，额外支持 Steam 5 位令牌。

## 功能列表
- 支持标准 TOTP 令牌
- 支持 Steam 5 位令牌
- 支持从其他 Steam Authenticator 迁移
- 支持 Steam 扫码登陆
- 支持 Steam 交易确认
- 支持 Steam 增强访问，解决部分网络环境无法访问 Steam 的问题
